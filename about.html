This website is made for the course Backend development at mid sweden university. The purpose of the website is to store your past work experiences. You can add new experiences on "add experiences" and you can find the saved work experiences on "your work experience". The website is using rest API with this database server: https://github.com/Rosalij/moment2backend.git. you can read more about the API in the README.
